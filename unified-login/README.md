#### 外部接口
* 路径: /verify
* 参数： 
    * id（Long） 平台id
    * username（String） 用户名
    * password(String) 原密码+平台令牌做MD51十六进制后的值
    * channel（Integer） 渠道：0:网页；1：安卓；2：ios
